Syllabus Goals

This course takes place during Fall 2024 semester at Hunter College as part of Hunter College's Emerging Media Studies major. The defined goals for this course are:

    Have a working understanding of the web’s history
    Have a working understanding ofcurrent authoring technologies including HTML 5, CSS 6, and JavaScript
    Create a media-rich website with a text editor that uses responsive design to adapt to the browser and mulitple devices
    Use appropriate technical vocabulary and pseudocode to think through problems and create site maps and wireframes to map out project development
    Apply JavaScript libraries and API’s for web development
    Distinguish important characteristics of Internet Art and identify leading practitioners
